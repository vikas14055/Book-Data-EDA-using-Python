# 📊 Book Data Exploratory Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on a book dataset using Python. It includes data cleaning, statistical analysis, and data visualization to gain meaningful insights from the dataset.

## Features
- Data Cleaning
- Price Conversion
- Summary Statistics
- Most Expensive Book
- Cheapest Book
- Average Price Calculation
- Missing Value Detection
- Duplicate Record Detection
- Hypothesis Testing
- Histogram Visualization

## Technologies Used
- Python
- Pandas
- Matplotlib

## Dataset
The dataset contains:
- Book Name
- Price

## Project Structure
```
Book-Data-EDA-using-Python/

│── README.md

```

## Installation

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python eda2.py
```

## Output
The project generates:
- Statistical Summary
- Price Analysis
- Missing & Duplicate Value Report
- Histogram of Book Prices
